# grpc-spring-kotlin

Example of gRPC server with Spring.

### Launch gRPC server

```
$ ./gradlew bootRun
```

Access with grpcc:

```
$ ./grpcc.sh
```

### Launch gRPC-Web proxy (optional)

```
$ docker-compose up -d
```
